# 📚 Book App

Vue 3 + Vite로 만든 도서 추천 앱입니다. 책을 검색하고, 카테고리별로 필터링하며, 즐겨찾기와 클릭 기반 추천 기능을 사용할 수 있습니다.

## ✨ 주요 기능

- 책 목록 조회
- 제목/저자 기반 검색
- 카테고리별 필터링
- 즐겨찾기 기능
- 상세 보기
- 클릭 기반 추천 도서 섹션

## 🛠 기술 스택

- Vue 3
- Vite
- Pinia
- Vue Router
- Bootstrap

## ▶️ 실행 방법

1. 의존성 설치
   ```bash
   npm install
   ```

2. 개발 서버 실행
   ```bash
   npm run dev
   ```

3. 브라우저에서 확인
   - 기본 주소: http://localhost:5173

## 📦 빌드

```bash
npm run build
```

## 🧩 프로젝트 구조

- src/components: 공통 UI 컴포넌트
- src/views: 홈, 상세, 즐겨찾기 화면
- src/stores: Pinia 상태 관리
- src/mocks: 목업 데이터
- src/api: 책 데이터 접근
